• Variant build_tileset_from_list(Array tilesets, String source_path, Dictionary options = {})

Builds a TileSet resource from a TSX file.

Reads a TSX file located at source_path and returns a TileSet resource based off of it. If there is an error, then an int is returned instead. tilesets is a list of tileset dictionaries in the format that would be returned by read_tileset_file().

You can modify the result of this function through the options dictionary. Below Is a list of options you can pass:
* columns = Limit the number of columns per row in the tileset (default: 0)
* margin = Add margin around the tileset in pixels (default: 0)
* spacing = Spacing between tiles in pixels (default: 0)
* imageflags = Flags from the enum Texture.Flags to set the image with (default: 0)

• int create_layer(Dictionary layer, Dictionary map_data)

Create a layer.

Returns a node corresponding to a type of layer being processed. layer is a Dictionary containing data about a layer.

map_data is a Dictionary with information about the whole tilemap that is needed to correctly build the layer. The following keys are recognized:

* tileset -- A TileSet resource used to build the tilemap
* cell_size -- A Vector2 specifying the size of a tile
* options -- A list of options inherited from build()

• Variant print_error(String err)

Prints an error.

• Variant read_tilemap_file(String path)

Reads a Tiled tilemap file.

Reads a TMX file pointed to by path. It can be either a .tmx file which case it is read as XML) or a JSON file. A dictionary containing information about the tileset is returned, or int upon failure.

If the returned value is an int, it will be a constant of enum Error.

• Variant read_tileset_file(String path)

Reads a Tiled tileset file.

Reads a TSX file pointed to by path. It can either be a .tsx file (in which case it is read as XML) or a JSON file. A dictionary containing information about the tileset is returned, or int upon failure.

If the returned value is an int, it will be a constant of enum Error.

• int validate_tileset(Dictionary tileset)

Validate a tileset.

Returns an integer from the enum Error. OK is returned if the tileset is valid, and ERR_INVALID_DATA otherwise.
