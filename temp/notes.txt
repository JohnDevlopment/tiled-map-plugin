read_tmx:
	open the file and read past all the non-element nodes
	first element must be 'map'
	get dictionary for 'map'
		https://doc.mapeditor.org/en/stable/reference/tmx-map-format/#map
	for each element to the end of the 'map'
		tileset: get tileset data and push to data->tilesets
			function: parse_tileset
		layer: parse tile layer data and push to data->layers
		group: parse group layer data and push to data->layers
		properties: parse properties and assign dict to data->properties
			assign property types to data->propertytypes

parse_tileset:
	called when a 'tileset' element is encountered by the parser
	break loop when </tileset> is encountered
	for each element under 'tileset'
		tile:
			get dictionary for 'tile'
				info: https://doc.mapeditor.org/en/stable/reference/tmx-map-format/#tile
				if 'properties' and 'propertytypes'
					data->tileproperties['id' as string] = 'properties'
					data->tilepropertytypes['id' as string] = 'propertytypes'
				data->tiles['id' as string] = dictionary
		image:
			get dictionary for 'image'
				info: https://doc.mapeditor.org/en/stable/reference/tmx-map-format/#image
				attribute 'source' in data->image
					error if undefined 'source'
				attribute 'width' in data->imagewidth
				attribute 'height' in data->imageheight
		property:
			get dictionary for 'property'
				info: https://doc.mapeditor.org/en/stable/reference/tmx-map-format/#tmx-property
			get property data and add to dictionary
				function: parse_properties
				data->properties = 'properties'
				data->propertytypes = 'propertytypes'
